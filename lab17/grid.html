Lab17: CSS Layout

Big Ideas

The subject of this lab is to get more familiar with CSS using more advacned CSS Layout Methods. In this lab we made more complex CSS layouts including Float Layout, Flexbox Layout, and Grid Layout.

Challenges

For the most part this lab was for the most part successful. It was kind of complicated trying to figure out how to code the new types of layouts. However, this lab wasn't too difficult and we didn't have hardly any issues.

Results

The results of this lab are displayed in this website. This lab was pretty successful and we were able to complete it fully. I think all of the different layouts look clean and well done to the best of my abilities.
